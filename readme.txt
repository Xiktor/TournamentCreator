Aby uruchomic projekt prosze wprowadzic sciezke do interpretera pythona

Zainstalowac pakiety zawarte w requirements.txt, tj.

	python -m pip install Django
	pip install numpy

Nastepnie mozna uruchomic projekt

	python manage.py runserver

W bazie sa juz dwa konta z przykladowymi danymi, konto administratora:
	login:	superUser
	haslo:	user12345
oraz konto zwyklego uzytkownika
	login:	user
	haslo:	user12345

Aplikacja nie wyglada zbyt estetycznie na przegladarce Explorer(problem z bootstrapem),
ja uzywalem Chrome.
